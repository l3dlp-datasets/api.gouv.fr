  L'API donne accès à tous les jeux linéaires du catalogue open data de l’Onisep. 
  L’Onisep collecte et produit principalement des données sur l’offre de formation initiale en France.
  Son catalogue open data propose, sur ce périmètre, des jeux de données sur : 

  - L’offre de formation
  - Les formations et certifications
  - Les structures d’enseignement et d’information
  - Les métiers
  - Le handicap

### À quoi sert l’API ?

  L’API permet de requêter directement tous les jeux linéaires présents au catalogue. Ces jeux comportent un onglet tableau et un onglet API facilitant l’exploration des contenus et les requêtes précises.
